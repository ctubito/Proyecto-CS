Penguin Species Classifier

My CS50 final project using machine learning.

About This Project

This project is a web application that classifies penguin species based on their physical measurements. It uses a machine learning model trained on the Palmer Archipelago penguin dataset.

The application also includes an unsupervised learning visualization using KMeans clustering to show how penguins can be grouped based on their physical characteristics.

KMeans Clustering

This plot shows how KMeans groups penguins based on their physical measurements. The colors represent the clusters found by the algorithm, while the marker shapes represent the real species labels.
